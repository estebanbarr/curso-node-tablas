# Notas:
Este es un programa pedorro en Node

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla que se va a mostrar
                                               [número] [requerido] [defecto: 1]
  -h, --hasta    Es el limite de las multiplicaciones, es decir hasta donde se  
                 va a multiplicar                         [número] [defecto: 10]
  -l, --listar   Indica si mostrar por pantalla las multiplicaciones
                                                     [booleano] [defecto: false]
```
